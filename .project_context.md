## ⚙️ Функционал
Приложение отображает статистику стримеров Twitch и каналов YouTube в реальном времени.

1. **Twitch (через DecAPI)**:
   * Запросы идут без ключей на публичный API: `https://decapi.me/twitch/...`
   * Парсятся данные: статус трансляции, фолловеры, зрители, аптайм, игра, возраст аккаунта, название стрима.
2. **YouTube (через YouTube Data API v3)**:
   * Требуется API-ключ, вводимый пользователем в настройках.
   * Получает ID канала, подписчиков, просмотры, количество видео и название последнего видео.
3. **Настройки**:
   * Хранятся локально на девайсе через `SettingsManager` (SharedPreferences).
   * Поддерживается динамическая смена темы (Светлая/Темная/Системная).
   * Поддерживается смена языка (Русский/Английский).

## 📂 Архитектурная структура
* **`com.m1kunyastudio.notstats.MainActivity`**: Точка входа, управление состояниями экранов (`setup`, `stats`, `settings`).
* **`com.m1kunyastudio.notstats.data`**:
  * `SettingsManager`: Локальное сохранение настроек и API ключей.
  * `TwitchRepository`: Логика запросов к Twitch.
  * `YouTubeRepository`: Логика запросов к YouTube Data API.
* **`com.m1kunyastudio.notstats.ui`**:
  * `StatsScreen`: Главный дашборд со вкладками Twitch и YouTube.
  * `SettingsScreen`: Экран редактирования ссылок и ключей, смены тем и языка.
  * `SetupScreen`: Стартовый экран для первой настройки.
