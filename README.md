# NotStats 📊

**NotStats** — это современное Android-приложение для отслеживания статистики Twitch-каналов и YouTube-каналов в реальном времени. Оно разработало на Kotlin с использованием современного инструментария **Jetpack Compose** и придерживается современных стандартов разработки под Android.

## 🚀 Основные возможности

* **Статистика Twitch (через DecAPI):**
  * Текущий статус стрима (Live / Offline)
  * Количество фолловеров (подписчиков)
  * Количество зрителей в реальном времени
  * Время аптайма текущей трансляции
  * Категория (игра) стрима
  * Возраст аккаунта
  * Название трансляции
* **Статистика YouTube (через YouTube Data API v3):**
  * Количество подписчиков канала
  * Общее количество просмотров
  * Общее количество загруженных видео
  * Название последнего опубликованного видео
* **Удобный UX/UI:**
  * Современный интерфейс на базе **Jetpack Compose** и компонентов **Material 3**.
  * Поддержка светлой, тёмной и системной тем оформления.
  * Мультиязычность (полная поддержка русского и английского языков с возможностью переключения внутри приложения).
  * Экран первоначальной настройки для быстрого старта.
* **Безопасность:**
  * Все API-ключи и настройки хранятся локально на устройстве пользователя через `SharedPreferences` и не коммитятся в репозиторий.

## 🛠️ Стек технологий

* **Язык программирования:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Архитектурный подход:** Clean Architecture (Repository Pattern)
* **Асинхронность:** Kotlin Coroutines & Flow
* **Сеть:** HttpURLConnection (для работы без тяжелых внешних библиотек)
* **Парсинг данных:** org.json

## 📦 Как запустить проект локально

### Требования
* Android Studio (последняя версия, например Ladybug или новее)
* JDK 17 или новее
* Android SDK (API level 24 или выше)

### Инструкция по сборке

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Funtyra52/notstats2.git
   ```
2. Откройте проект в **Android Studio**.
3. Дождитесь окончания синхронизации Gradle.
4. Подключите физическое устройство через USB (с включенной отладкой по USB) или запустите эмулятор.
5. Нажмите кнопку **Run** в Android Studio или соберите проект через терминал:
   ```bash
   ./gradlew installDebug
   ```

## 🔑 Настройка YouTube API Key

Для работы функции отслеживания YouTube-статистики необходим ключ доступа к YouTube Data API v3:

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/).
2. Создайте новый проект (или выберите существующий).
3. Перейдите во вкладку **Library** (Библиотека API), найдите **YouTube Data API v3** и включите его.
4. Перейдите в раздел **Credentials** (Учетные данные) и нажмите **Create Credentials** -> **API Key**.
5. Скопируйте полученный ключ.
6. Вставьте ключ в настройках приложения **NotStats** в соответствующее поле.
